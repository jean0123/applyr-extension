Data & Privacy

Manage your stored data and privacy settings.

🔒 Privacy

Applyr stores all data locally in Chrome's encrypted storage. Your API keys are encrypted using AES-256-GCM. Resume text and profile data are never sent to Applyr's servers — only to the AI provider you choose.

✓ No account required · ✓ No data collection · ✓ Local-first
