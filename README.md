# Applyr – AI Resume Tailoring & Auto-Apply Chrome Extension

Applyr is a Chrome extension that automatically tailors your resume to match a specific job posting on **Indeed** or **LinkedIn**, then auto-fills the application form with your customized resume.

---

## Features

- **AI-powered tailoring** — matches your experience to the job's requirements using Claude, ChatGPT, or Gemini
- **One-click apply** — injects a floating "Tailor & Apply" button next to each job's Apply button
- **Resume preview** — review AI changes before submitting
- **Auto-upload** — uploads your tailored PDF resume directly into the application form
- **Application history** — keeps a log of every job you've applied to
- **Encrypted storage** — API keys are AES-256-GCM encrypted locally; nothing is sent to Applyr's servers
- **Free tier** — bring your own API key (Claude, ChatGPT, or Gemini)
- **Free forever** — bring your own API key, no subscription required

## Installation

### 1. Clone or download this repository

```bash
git clone https://github.com/jean0123/applyr-extension.git
# or download and unzip
```

### 2. Generate the icons (one-time setup)

```bash
node generate-icons.js
```

This creates `assets/icon16.png`, `assets/icon48.png`, and `assets/icon128.png`.

### 3. Load the extension in Chrome

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `applyr-extension` folder
5. The Applyr icon (indigo "A") will appear in your toolbar

## Setup

### Upload Your Resume

1. Click the **Applyr icon** in your Chrome toolbar to open the popup
2. In the **"Your Resume"** section, drag & drop your PDF resume, or click to browse
3. Applyr extracts and stores the text locally

**Alternative:** Go to **Settings → Resume** and paste your resume text directly (useful if your PDF doesn't parse correctly).

### Configure Your AI Provider

1. In the popup (or **Settings → AI Configuration**), select your AI provider:
   - **Claude** — requires an API key from [console.anthropic.com](https://console.anthropic.com) — Recommended
   - **ChatGPT** — requires an API key from [platform.openai.com](https://platform.openai.com)
   - **Gemini** — requires an API key from [aistudio.google.com](https://aistudio.google.com)
2. Paste your API key into the field and click **Save**

Your API key is encrypted using AES-256-GCM and stored only in Chrome's local storage. It is never sent to Applyr's servers.

### Fill Your Profile (Optional)

Go to **Settings → Profile** and fill in your name, email, phone, and location. This information can be used to pre-fill application form fields.

## Usage

1. Navigate to a job posting on **Indeed** (`indeed.com`) or **LinkedIn** (`linkedin.com/jobs`)
2. A **"Tailor & Apply"** button (indigo, rounded) will appear next to the native Apply button
3. Click **"Tailor & Apply"** — the Applyr panel slides in from the right
4. Review the status (resume loaded ✓, AI configured ✓)
5. Click **"✦ Tailor My Resume"** — the AI rewrites your resume for this specific job (takes ~10–30 seconds)
6. Review the **"Key Changes Made"** summary and optionally download your tailored PDF
7. Click **"Apply Now →"** — Applyr clicks the native Apply button and uploads your tailored resume

## Troubleshooting

### "Tailor & Apply" button doesn't appear
- Make sure you're on a **job detail page** (not a search results page)
- On Indeed, click into the specific job listing
- On LinkedIn, click "See more" to expand the full job description
- Refresh the page and wait 2–3 seconds

### "Could not extract text from PDF"
- Some PDFs are image-based (scanned) and can't be parsed without OCR
- Go to **Settings → Resume** and paste your resume text directly
- Or export your resume from Word/Google Docs as a plain-text PDF

### "No API key found" error
- Open the Applyr popup and check your provider and API key
- Make sure you saved the key (click Save or press Enter)
- Verify the key is valid by testing it in the provider's own playground

### API errors (401, 403)
- Your API key is invalid or expired — generate a new one
- Claude: ensure you have available credits on your Anthropic account
- OpenAI: ensure billing is set up on platform.openai.com

### Rate limit errors (429)
- You've hit the API rate limit — wait 30–60 seconds and try again
- Consider upgrading your API tier for higher rate limits

### LinkedIn "Easy Apply" not found
- Some LinkedIn jobs require applying on the company website (no Easy Apply button)
- In this case, Applyr will still tailor your resume — download it and apply manually

### The upload didn't work
- Some application systems use custom file upload components that block automated uploads
- Download your tailored resume PDF and upload it manually

## Privacy & Security

- All data is stored locally in Chrome's `chrome.storage.local`
- API keys are encrypted with **AES-256-GCM** using the Web Crypto API
- Your resume text and profile are **never** sent to Applyr's servers
- Data flows only between your browser and the AI provider you select
- You can delete all stored data from **Settings → Data & Privacy → Reset Everything**
